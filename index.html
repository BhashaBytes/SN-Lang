SN-Lang: Sanskrit-Inspired Programming

Welcome to SN-Lang – a programming language designed especially for Sanskrit speakers! SN-Lang makes coding more intuitive by using Sanskrit words and their transliterations as keywords, instead of English-based Python syntax. It’s beginner-friendly and runs fully on Python via a translation layer.

Write code below, run it instantly, and explore SN-Lang’s features with live examples. Learning programming has never been this friendly!

Documentation & Examples

Print (दर्शय / darshaya)

दर्शय("Hello World")

Functions (निरूपय / nirupaya)

निरूपय योगः(a, b):
    निवर्तय a + b
दर्शय("फलम् =", योगः(5, 3))

Input (गृहय / grahaya)

a = int(गृहय("संख्या १:"))
b = int(गृहय("संख्या २:"))
दर्शय("योगः =", a + b)

Conditionals

i = 5
यदि i > 3:
    दर्शय("i बृहत्")
अन्यदि i == 3:
    दर्शय("i समम्")
अन्यथा:
    दर्शय("i लघु")

Loops

अर्थम् x in [1,2,3]:
    दर्शय("x =", x)
i = 0
यावत् i < 3:
    दर्शय("i =", i)
    i += 1

Loop Control

अर्थम् x in [1,2,3]:
    यदि x == 2:
        दर्शय("द्वितीयम् प्राप्तम्")
        विच्छेद
    अन्यथा:
        दर्शय(x)

Booleans & None

यदि सत्य:
    दर्शय("true")
अन्यदि मिथ्या:
    दर्शय("false")
दर्शय("शून्यम् =", शून्य)

Exceptions

प्रयत्न:
    1 / 0
विना ZeroDivisionError:
    दर्शय("शून्येन भागः न शक्यः")
अन्ततः:
    दर्शय("एतत् नित्यं भवति")

Imports

आयातय math रूपेण m
दर्शय(m.sqrt(4))
